Start

Use start or start- to add an offset. start=1 can be used to force a new row.
start=7 starts in the middle, since it's 1 based.
If that's too much classes for you, you can disable their generation by setting $grd-start: false.

start="1" start-md="3"
col="4" start="7" start-lg="9"
col="2" start="1"

Prebuilt smart grids

Use grd-2, grd-3 or grd-4 for smart responsive grids. They fully collapse on the smallest breakpoint.
If that's too much classes for you, you can disable their generation by setting $grd-smart-grids: false.
